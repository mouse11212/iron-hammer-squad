### 真实语法进阶图景（据教材实际抽取，非课标假设）

| 学段 | 真实主线 |
|---|---|
| 三上 L1 | be 动词 + 简单句（问候/物品/颜色/数字/家庭/方位） |
| 三下 L1 | have got · like/like + V-ing · 报时 · 星期与科目 |
| 四上 L1 | **can/can't 能力** · may 许可 · 天气/四季/交通/问路（祈使句） |
| 四下 L1 | **现在进行时**（What are you doing/wearing）+ be/feel · can · there-be |
| 五上 L2 | should · 频度副词 · 连词 · **将来时 be going to / will** |
| 五下 L2 | **一般过去时**（was/were · -ed · 不规则动词 · Did） |
| 六上 L2 | **比较级 / 最高级** · How much · 顺序词 · should · 感叹句 |
| 六下 L2 | 祈使句 · 连词 · **多时态综合**（过去+现在+将来） |

### ⚠️ 据实修正（与课标级别先验假设的出入，红线1 真值优先）

1. **四上**：先验设「现在进行时起步」→ 实际**无进行时**，主线是 can/can't 能力；进行时其实在**四下**才成为主线。
2. **五上**：先验设「一般过去时起步」→ 实际是**将来时（be going to / will）**。
3. **顺序倒置**：本套三起教材**先教将来时（五上）再教过去时（五下）**，与"先过去后将来"的常见直觉相反——游戏按年级注入语法范围时须以此真实顺序为准。
