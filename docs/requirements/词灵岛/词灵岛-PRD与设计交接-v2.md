## 1. 产品概念与业务场景

把外研版小学英语教材(3–6 年级全程)做成一座可探索的漫画大陆:孩子在 15 分钟一局的闯关里收服"词灵"(单词/句型),并能和一个**只说"学过的话"(按年级)**的 AI 词灵伙伴做真实英文对话——把应试知识点变成有兴趣、可坚持、家长放心的游戏化练习。

痛点(学生:机械记忆易忘、应用弱、缺兴趣;家长:要效果又要控屏护眼;行业:缺低门槛"开口说"陪练)与"为什么用 AI(curriculum-bounded 口语陪练)"同 v1,不再赘述。

## 3. 教材知识来源与语法进阶（覆盖 3–6 年级）

**权威依据（v2.1 更正）**:本节语法主干与年级落点,以**教材知识库 `教材/`(外研三起·孙有中新版,8 册逐单元抽取,`source_tag=verified-sanqi`)**为准;课标(2022)级别划分(一级 3-4 / 二级 5-6)仍适用。**据实提醒**:本套三起教材语法进阶顺序与课标先验假设有出入(见 KB §据实修正),下表已按 KB **真实抽取顺序**更正,不再按"一起"假设。

**世界 = 3–6 年级分级大陆;语法焦点锚定课标级别:**

| 年级 | 课标级别·语法主干(权威) | 岛屿示例(主题) | 代表目标句型 |
|---|---|---|---|
| 三年级 | **一级**:be 动词/简单句(上)· have got/like+V-ing/报时/星期科目(下) | 问候岛 / 彩虹森林(颜色)/ 数字山谷 / 家庭小镇 | I'm…/ What's your name? / How old are you? / Have you got…? |
| 四年级 | **一级**:**can/can't 能力·may 许可·祈使句(上)** · **现在进行时(下)** | 能力擂台 / 天气车站 / 进行时剧场 / 问路集市 | Can you…? / It's sunny. / She's reading a book. / Where's…? |
| 五年级 | **二级**:should/频度副词/连词·**将来时 be going to/will(上)** · **一般过去时(下)** | 目标未来站 / 作息钟楼 / 往事回廊(过去时)/ 节日庙会 | I'm going to… / will… / What did you do? — I played… |
| 六年级 | **二级**:**比较级/最高级·How much·感叹句(上)** · 祈使句/连词/多时态综合(下) | 比较擂台 / 理财市场 / 旅行规划 / 故事剧场 | taller than… / How much…? / What a…! |

> 课标 2022 级别划分(一级/二级)适用,但**本套三起教材实际语法进阶顺序以 KB 为准**(非课标表11 先验顺序):进行时在**四下**、将来时在**五上**、过去时在**五下**(先将来后过去)。详见 `教材/README.md` §真实语法进阶图景 + §据实修正(红线1 真值优先)。

**应试对齐**:按课标听/说/读/写 + 小学常见题型(听音选词、看图选词、句型连线、补全句子、跟读评测)设计题型池,**按年级进阶**:三四年级以听音/看图选词为主;五六年级加入句型连线、补全、跟读评测,匹配二级"理解结构、描述比较事件"的要求。

**录入规范**:每个题目挂 `年级 / 课标级别 / 语法结构 / 目标句型 / 四技能标签 / 题型 / 教材落点`,`source` 标 `verified-sanqi`(三起·孙有中逐单元已核实)。原"待一起核对"校验位作废(v2.1：三起即权威源,无需回"一起"对齐)。

**版权合规**:不复制教材原文/插图;仅以词汇、句型功能、知识点为索引,语料与插画原创。

## 5. 功能设计(8 大模块)

与 v1 相同的 8 模块,按 3–6 年级做如下强化:
1. **登录与家长守护**:登录后**选年级**;家长守护(每日 15 分钟、报告、实物兑换确认)为硬约束。
2. **冒险地图(分级闯关)**:**年级选择器**切三/四/五/六;每年级一片大陆,岛屿=主题,语法焦点随年级进阶(锚定课标级别)。
3. **关卡玩法**:题型库**按年级进阶**(三四:听音/看图选词;五六:加句型连线/补全/跟读)。
4. **AI 英文对话**(见 §7):Robo **按年级**限定可用词汇与句型(三四:现在时/can/进行时(四下);五六:将来时(五上)/过去时(五下)/比较级)。
5. **积分制**:⭐星星 + 🪙金币 + 连续登录/连胜;可配置防刷。
6. **礼物兑换**:虚拟装扮/坐骑/贴纸 + 实物奖励(家长确认)。
7. **番茄钟 15 分钟**(见 §6):每局硬上限,温和落幕,护眼。
8. **成长档案**:**分年级**记录星星、徽章、已掌握知识点(按课标级别/四技能)、学习时长;孩子成就视图 + 家长应试报告视图。

## 6. 番茄钟时长机制

每局 = 1 个 15 分钟番茄钟,界面常驻"🍅 能量钟";到点当前题做完后进入"今天的冒险结束啦"结算页(护眼/休息正向引导,非惩罚);家长可配每日番茄数上限与冷却。这是家长付费的关键信任锚点,契合双减与未成年人健康用网导向。同 v1。

---

## 7. AI 对话功能设计与护栏(按年级)

定位:curriculum-bounded 口语陪练,**非开放聊天**。
- 系统提示**按年级**约束模型只用对应级别已学词汇与句型(据 KB 真实进阶):**三年级**=be 动词/have got/like+V-ing/一般现在时;**四年级**=+can/can't·may·祈使句(上)、现在进行时(下);**五年级**=+将来时 be going to/will(上)、一般过去时(下);**六年级**=+比较级/最高级·多时态综合。难度随关卡进度解锁,**按真实顺序(先将来后过去)**。
- 多模态(文本+语音+发音评测)、脚手架(快捷气泡、"💡帮我说"、中文小注)同 v1。
- **儿童安全护栏为最高优先级(硬性)**:话题白名单+输出过滤;不索取真实隐私;不诱导延长使用/情感依赖;全程家长可见;异常留痕告警;模型不可超纲(每轮经知识点库按年级校验)。

## 10. Claude Code 开发交接规格

**技术建议**:React + Vite + Tailwind(token 写入配置);自建漫画组件;Node/TS API;PostgreSQL;AI 对话经 Claude Agent SDK 注入"年级知识范围"系统提示 + 输出护栏;语音 ASR/TTS + 发音评测。同 v1。

**屏清单(见 `wordspirit-prototype-v2.html`,6 屏)**:登录(含选年级)→ 冒险地图(**年级选择器 + 语法焦点字幕**)→ 关卡玩法 → AI 对话 → 礼物商店 → 番茄钟结束。

**组件清单**:在 v1 基础上新增 `GradeSelector`(年级切换)、`GradeBandCaption`(语法焦点字幕)。其余 `StickerCard/SpeechBubble/ChunkyButton/Pill/MapNode/QuizOption/ChatMessage/ScaffoldChip/ShopItem/TomatoClock/TabBar` 同 v1。

**核心数据模型(草案,v2 增加年级维度)**
```
User(id, nickname, grade∈{3,4,5,6}, parentGuard{dailyTomatoLimit, cooldown})
Region(id, grade, cefrLevel∈{L1,L2}, moduleRef, title, order, unlockRule)
Level(id, regionId, grade, order, starRule, questionIds[])
KnowledgePoint(id, grade, cefrLevel, module, unit, type[word|pattern],
               grammar[be|present-simple|present-continuous|past|future|comparative|...],
               skills[L/S/R/W], text, source[verified|need-yiqi-check])
Question(id, levelId, kpId, format[listen-pick|img-pick|match|cloze|speak], payload, answer)
Progress(userId, levelId, grade, stars, masteredKpIds[])
Wallet(userId, coins, stars)
ShopItem(id, kind[avatar|sticker|real], price, requiresParent)
ChatSession(userId, gradeScope, grammarScope[], messages[], flagged?)
PlaySession(userId, startedAt, tomatoSeconds=900, endedReason)
```
**关键约束**:① 题目挂 `kpId` 且 `KnowledgePoint` 带 `grade/cefrLevel/grammar/source`(应试覆盖度可按年级统计);② AI 对话每轮按 `gradeScope/grammarScope` 校验不超纲;③ `PlaySession` 900 秒硬上限;④ 实物兑换 `requiresParent=true`;⑤ 知识点 `source=verified-sanqi`(三起·孙有中逐单元已核实)即可上线;v2.1 起不再回"一起"教材核对。

**验收(部分 DoD)**:最小切片闭环跑通;番茄钟 900 秒温和落幕;AI 对话不输出超纲词且通过儿童安全过滤;所有题目可追溯到课标级别与教材主题,`need-yiqi-check` 项不得在未核对状态下进入正式题库。
